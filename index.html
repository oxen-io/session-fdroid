Session F-Droid Repository

This is the Official Session F-Droid repo for Session Android, add it to your F-Droid client to receive Session F-Droid updates directly to your device. You will need to have F-Droid installed before proceeding with the following steps.

Add Manually

Instructions

Open
FDroid > Settings > Repositories > + button
 then copy & paste the repository URL and fingerprint. This will add the Session repository, which means the app will now come up on the store.
